GitHub & Deployment Instructions

Push to GitHub

# Initialize git repository
git init

# Add all files
git add .

# Commit initial setup
git commit -m "Initial D.N Express Logistics website with brand identity and API integration"

# Add remote origin (replace with your GitHub repo URL)
git remote add origin https://github.com/YOUR_USERNAME/dnexpress-logistics.git

# Push to GitHub
git branch -M main
git push -u origin main

GitHub Repository Setup

  1. Create new repository on GitHub: dnexpress-logistics
  2. Add description: "D.N Express Logistics - Global shipping platform"
  3. Set repository to Public for sharing
  4. Enable GitHub Pages for live demo
  5. Add README.md with project details

GitHub Pages Setup

  1. Go to repository Settings ‚Üí Pages
  2. Set source branch to main
  3. Select folder: root
  4. Your site will be available at: https://YOUR_USERNAME.github.io/dnexpress-logistics/
